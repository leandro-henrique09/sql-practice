# 📊 SQL para Análise e Engenharia de Dados

Este repositório reúne meus estudos práticos em SQL com foco em aplicação para Análise e Engenharia de Dados.

Aqui você encontrará consultas, exercícios, desafios e simulações de cenários reais de negócio, organizados por nível e tema.

---

## 🎯 Objetivo

Desenvolver domínio em:

- Consultas SQL avançadas
- Manipulação e transformação de dados
- Joins e modelagem relacional
- Subqueries e CTEs
- Window Functions
- Agregações e métricas de negócio

---

## 🚀 Próximos Passos

- Construção de projetos com datasets reais
- Modelagem dimensional
- Integração com pipelines em Python
- Aplicação em cenários de Data Engineering
